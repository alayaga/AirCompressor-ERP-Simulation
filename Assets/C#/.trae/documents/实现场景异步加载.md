# 实现场景异步加载

## 目标
修改`SceneManage.cs`中的`GoToMainScene`方法，添加异步加载功能，并使用加载场景显示加载进度。

## 实现步骤

### 1. 修改`SceneManage.cs`文件
- 将`GoToMainScene`方法改为异步方法（使用`async`关键字）
- 使用`SceneManager.LoadSceneAsync`替代同步加载
- 添加加载场景的过渡逻辑

### 2. 实现异步加载逻辑
```csharp
public async void GoToMainScene()
{
    // 1. 加载加载场景
    SceneManager.LoadScene("加载场景");
    
    // 2. 异步加载主场景
    AsyncOperation asyncOperation = SceneManager.LoadSceneAsync("主场景");
    asyncOperation.allowSceneActivation = false;
    
    // 3. 等待加载完成
    while (!asyncOperation.isDone)
    {
        // 4. 可以在这里更新加载进度（例如通过事件或单例）
        float progress = Mathf.Clamp01(asyncOperation.progress / 0.9f);
        
        // 5. 当加载进度达到0.9时，允许场景激活
        if (asyncOperation.progress >= 0.9f)
        {
            asyncOperation.allowSceneActivation = true;
        }
        
        await Task.Yield();
    }
}
```

### 3. 添加必要的命名空间
- 需要添加`using System.Threading.Tasks;`以支持异步操作

### 4. 加载场景要求
- 项目中需要有一个名为"加载场景"的场景，用于显示加载进度
- 加载场景可以包含进度条、加载动画等UI元素

## 预期效果
- 调用`GoToMainScene`时，首先切换到加载场景
- 在后台异步加载主场景
- 加载完成后自动切换到主场景
- 支持显示加载进度

## 注意事项
- 确保"加载场景"和"主场景"在Build Settings中已添加
- 如果需要在加载场景中显示进度，可能需要实现一个单例或事件系统来传递进度信息
- 异步方法需要Unity 2017.2或更高版本支持
